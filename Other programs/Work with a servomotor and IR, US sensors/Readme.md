# Работа с сервомотором, ультразвуковым датчиком расстояния и инфракрасным датчиком движения

Задание 1 Работа с сервомотором sg90
Для начала работы я создал схему подключения сервомотора к платформе. Затем я приступил к созданию 
скетча. Задание делилось на 2 скетча: первый просто крутил лопасти мотора на заданные в скетче градусы, 
а второй крутился на градус, который отправлялся в Arduino из монитора порта. Так задание требует 
подключение библиотеки «Servo.h» .

Задание 2 Подключение ультразвукового датчика расстояния HC SR04
Для того чтобы начать работу, я составил в Fritzing. После создания схемы, я 
приступил к написанию скетча. Принцип работы заключался в том, что при попадании в поле зрения датчика 
какого-нибудь препятствия, то платформа отправляла бы точное значение в монитор порта в сантиметрах.
Также в этом задании надо было экспериментально определить угол, при котором ультразвуковой датчик 
начинает видеть барьер. Я его определил, угол равен 30 градусам.

Задание 3 Подключение инфракрасного датчика движения
Первым делом я создал схему. Далее надо было создать свой скетч, смысл которого в том, что датчик, 
улавливая какие либо движения в радиусе нескольких метров должен был выдавать в мониторе порта либо 1, 
если есть движение, либо нет, если его нет.

Чтобы проверить самостоятельно, как работают программы, Вам нужно:
1) Скачать и установить Arduino IDE;
2) Соберите цепь как показано на рисунках с названием m.png соответственно для каждой работы;
3) Подключите плату Arduino к компьютеру через USB-вход с помощью USB 2.0 Cable Type A/B;
4) Запустите Arduino IDE через скетч (файл с расширением .ino) из соответствующей папки;
5) Выберите на панели Select Board Ваш порт(Arduino Uno COM3);
5) Нажмите кнопку "Upload".
